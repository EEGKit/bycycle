Fitting Oscillations & One-Over F¶

bycycle is a python implementation of a cycle-by-cycle approach to analyzing neural oscillations (Cole & Voytek, 2018).

If you’re just getting started, you might want to start with the Readme, which also contains installation instructions.

If you want to explore how everything works, check out the Tutorial (needs to be created and linked like readme).
